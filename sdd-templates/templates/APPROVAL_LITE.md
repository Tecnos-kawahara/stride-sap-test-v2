## Lite Mode について

Lite Mode は小規模プロジェクト・PoC・1人開発向けの簡略化された承認フローです。
6段階の承認を3段階に削減しています。

| Full Mode | Lite Mode | 対象ファイル |
|-----------|-----------|-------------|
| Gate 1 + Gate 2 | Gate A | basic_design.md, process.bpmn |
| Gate 3 + Gate 4 | Gate B | spec.md, plan.md |
| Gate 5 + Final | Gate C | tasks.md, 実装, 検証 |
